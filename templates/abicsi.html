Abicsi.com


Abicsi Sunglasses.

Type:

Web Development

Year:

2022

Drawing inspiration from handmade hobby weather websites this site was written in vanilla JavaScript, HTML & CSS. Fetching UVI (Ultraviolet Index) data through an asynchronous API call to Weatherapi.com.. To create the front page loading animation, custom .NetCDF data from a full year of global UVI levels were plotted on a map and turned into a GIF. Provided by Temis.com. Payment processing and Inventory management handled through Shopify.

  • JavaScript, HTML & CSS
  • Asynchronous API request to retreive UVI weather data from Weatherapi.com.
  • Custom .NetCDF map data retreived from Temis.com
  • Images stored on IPFS.
  • Payment processing and Inventory management handled through Shopify.
